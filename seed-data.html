Popular Banco de Dados com Dados de Exemplo

Clique no botão abaixo para adicionar um conjunto de usuários de exemplo à sua coleção 'usuarios' no Firestore. Isso permitirá que você teste as funcionalidades da página de gestão de usuários.

Aviso: Execute este processo apenas uma vez. Executar múltiplas vezes irá sobrescrever os dados com os mesmos valores.
